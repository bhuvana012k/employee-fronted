# Employee Management System Frontend

As part of my learning journey in Java Full Stack Development, I developed the frontend of an Employee Management System separately using HTML, CSS,JavaScript and Reacts.

## Features

* Employee Registration Form
* Employee Login From
* Employee Details Display
* Responsive User Interface
* Form Validation
* User-Friendly Design

## Technologies Used

* HTML
* CSS
* JavaScript
* Reacts

## Project Purpose

This project was developed to improve my frontend development skills and understand user interface design concepts in web development.
